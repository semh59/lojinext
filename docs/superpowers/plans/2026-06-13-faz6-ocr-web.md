# Faz 6 — OCR Web Entegrasyonu Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans. Checkbox (`- [ ]`) adımları.

**Goal:** Web'den fiş fotoğrafı yükle → OCR önizleme → kullanıcı onayı → `yakit_alimlari` kaydı; + yakıt belgesi arşiv görünümü. (Telegram→OCR→sefer_belgeler akışı zaten var; bu fazda web tarafı eklenir.)

**Architecture:** (1) Backend OCR bridge: `POST /fuel/ocr-preview` — yüklenen görseli mime+magic-byte doğrula → `get_monitored_client` ile `OCR_SERVICE_URL/ocr/process`'e ilet → yapılandırılmış sonucu (litre/tutar/km/tarih/istasyon) döndür (DB yazmaz, önizleme). (2) Belge arşivi: `GET /fuel/documents` — `sefer_belgeler` (yakit_fisi) listesi. (3) Frontend: `ReceiptUpload` bileşeni — dosya seç → ocr-preview → düzenlenebilir form (araç seç) → mevcut `fuelService.create` ile kayıt; FuelPage'e eklenir.

**Tech Stack:** FastAPI, httpx (get_monitored_client), React + React Query, pytest, vitest.

**Önkoşullar (kod doğrulandı 2026-06-13):**
- OCR servisi: `POST {OCR_SERVICE_URL}/ocr/process` (files={file}, data={belge_tipi}) → `{"ham_metin": str, "yapilandirilmis": {litre, tutar, km, tarih, istasyon}}`. `settings.OCR_SERVICE_URL="http://ocr-service:8001"`.
- `from app.infrastructure.monitoring.external_api_probe import get_monitored_client` → `async with get_monitored_client(timeout=) as client: await client.post(...)` (ocr_tasks.py:49 deseni).
- Magic-byte: `app/api/v1/endpoints/internal.py` `_looks_like_allowed_image(bytes)->bool` + `_ALLOWED_MIME_TYPES` (image/jpeg|png|webp). Yeniden kullan (DRY).
- `POST /fuel/` (create_yakit, **admin**) zaten `YakitCreate` ile kayıt oluşturur — OCR onayından sonra frontend bunu kullanır. `fuelService.create` (frontend) mevcut.
- `YakitCreate` (YakitBase + toplam_tutar): arac_id, tarih, istasyon, fiyat_tl, litre, toplam_tutar, km_sayac, fis_no... OCR map: litre→litre, tutar→toplam_tutar, km→km_sayac, tarih→tarih, istasyon→istasyon.
- `SeferBelge` (sefer_belgeler): id, sofor_id, sefer_id, belge_tipi, dosya_yolu, ocr_ham, ocr_veri(JSONB), ocr_durumu, created_at.
- Fuel router prefix `/fuel` (api.py). FuelPage: feature bileşenleri (FuelHeader/FuelTable/FuelModal...). fuel-service: getAll/create/update/delete.
- Lokal faithful test: bkz [[local-test-db-execution]].

```python
@router.post("/ocr-preview", response_model=OcrPreviewResponse)
async def ocr_preview(
    file: UploadFile = File(...),
    _user=Depends(get_current_active_user),
) -> OcrPreviewResponse:
    """Fiş görselini OCR servisine iletip yapılandırılmış önizleme döndürür.

    DB'ye yazmaz — kullanıcı önizleyip onayladıktan sonra POST /fuel/ ile
    kayıt oluşturur. Görsel mime + magic-byte ile doğrulanır (ARCH-020).
    """
    content_type = file.content_type or ""
    if content_type not in _ALLOWED_MIME_TYPES:
        raise HTTPException(
            status_code=415, detail=f"Desteklenmeyen dosya türü: {content_type}"
        )
    content = await file.read()
    if not _looks_like_allowed_image(content):
        raise HTTPException(
            status_code=415, detail="Geçerli bir görsel dosyası değil"
        )
    try:
        async with get_monitored_client(timeout=90) as client:
            resp = await client.post(
                f"{settings.OCR_SERVICE_URL}/ocr/process",
                files={"file": ("fis.jpg", content, content_type)},
                data={"belge_tipi": "yakit_fisi"},
            )
        resp.raise_for_status()
        result = resp.json()
    except HTTPException:
        raise
    except Exception as exc:
        logger.error("OCR servisi hatası (web preview): %s", exc)
        raise HTTPException(
            status_code=502, detail="OCR servisi şu an yanıt vermiyor"
        )
    yap = result.get("yapilandirilmis") or {}
    return OcrPreviewResponse(
        ham_metin=result.get("ham_metin"),
        yapilandirilmis=OcrParsedFields(
            litre=yap.get("litre"),
            tutar=yap.get("tutar"),
            km=yap.get("km"),
            tarih=yap.get("tarih"),
            istasyon=yap.get("istasyon"),
        ),
    )
```

```typescript
import { useState } from "react";
import { fuelService, type OcrPreview } from "../../services/api/fuel-service";

interface Props {
  onConfirm: (fields: OcrPreview["yapilandirilmis"]) => void;
}

export function ReceiptUpload({ onConfirm }: Props) {
  const [fields, setFields] = useState<OcrPreview["yapilandirilmis"] | null>(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  const onFile = async (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0];
    if (!file) return;
    setLoading(true);
    setError(null);
    try {
      const res = await fuelService.ocrPreview(file);
      setFields(res.yapilandirilmis);
    } catch {
      setError("OCR okunamadı; alanları elle girebilirsiniz.");
      setFields({ litre: null, tutar: null, km: null, tarih: null, istasyon: null });
    } finally {
      setLoading(false);
    }
  };

  const set = (k: keyof NonNullable<typeof fields>, v: string) =>
    setFields((f) => (f ? { ...f, [k]: v } : f));

  return (
    <div className="rounded-modal border border-border bg-surface p-4 space-y-3">
      <h3 className="text-sm font-semibold text-secondary">Fiş Yükle (OCR)</h3>
      <label className="block text-sm text-tertiary">
        Fiş fotoğrafı
        <input
          type="file"
          accept="image/*"
          aria-label="Fiş fotoğrafı"
          onChange={onFile}
          className="mt-1 block"
        />
      </label>
      {loading && <p className="text-sm text-tertiary">OCR okunuyor…</p>}
      {error && <p className="text-sm text-warning">{error}</p>}
      {fields && (
        <div className="space-y-2">
          {(["istasyon", "litre", "tutar", "km", "tarih"] as const).map((k) => (
            <label key={k} className="flex items-center justify-between gap-2 text-sm">
              <span className="text-tertiary capitalize">{k}</span>
              <input
                value={fields[k] ?? ""}
                onChange={(e) => set(k, e.target.value)}
                className="rounded-card border border-border bg-elevated px-2 py-1"
              />
            </label>
          ))}
          <button
            type="button"
            onClick={() => fields && onConfirm(fields)}
            className="rounded-card bg-elevated px-3 py-1 text-sm text-primary"
          >
            Onayla ve kaydet
          </button>
        </div>
      )}
    </div>
  );
}

export default ReceiptUpload;
```
